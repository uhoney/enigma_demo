Detailed Description

Represents a rotor in the Enigma machine, responsible for character substitution based on its wiring and position.

Notes: the actual rotor did not have an index[0], computers do. Rotor always spins first, so rotor_index will always be index[n+1] before any actions. You take your character (i.e 'Z') which is on alphabet_index[25] and add rotor_index to it and take mod 26. so you get (with rotor_index 0, char 'Z') 1+25 = 26 mod 26 = 0. This is the index of char to output, which is 'E'.
